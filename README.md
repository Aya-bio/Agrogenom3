# AgroGenome AI

**AI-Powered Future of Agriculture & Livestock**

A modern, professional web platform that combines Agriculture, Biotechnology, AI/Machine Learning, and Business Management into one smart ecosystem for farmers, agribusinesses, and livestock owners.

## 🌟 Features

### Core Modules

- **🏠 Landing Page** - Premium marketing page with hero section, benefits, testimonials, and pricing
- **📊 Dashboard** - Real-time farm metrics and AI recommendations
- **🌱 Seed Intelligence** - AI-powered seed recommendations based on soil and climate data
- **🐄 Livestock Health AI** - Disease detection and health monitoring using computer vision
- **📈 Business Analytics** - Comprehensive charts and data-driven insights
- **🛒 Marketplace** - Agricultural products marketplace
- **ℹ️ About Us** - Company mission, vision, and team
- **📧 Contact** - Contact form and information

### Advanced Features

- ✅ Dark Mode / Light Mode
- ✅ Arabic / English Language Support (RTL)
- ✅ Fully Responsive Design
- ✅ Interactive Charts (Recharts)
- ✅ Smooth Animations (Motion)
- ✅ Modern UI/UX Design
- ✅ AI Simulation & Mock Data

## 🛠️ Tech Stack

- **Frontend Framework:** React.js 18.3.1
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Routing:** React Router v7
- **Charts:** Recharts
- **Animations:** Motion (Framer Motion)
- **Icons:** Lucide React
- **Build Tool:** Vite

## 🚀 Getting Started

### Prerequisites

- Node.js 16+ or Bun
- pnpm (recommended) or npm

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/agrogenome-ai.git

# Navigate to project directory
cd agrogenome-ai

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

### Build for Production

```bash
# Create production build
pnpm build

# Preview production build
pnpm preview
```
